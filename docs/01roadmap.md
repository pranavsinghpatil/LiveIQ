# [RESEARCH] Roadmap

## Vision
To build a research-first agentic system that scouts hypotheses, critiques them, and iteratively evolves research insights.

## Phases
1. **Foundation Setup** (Week 1–2)
   - Repo structure, docs, baselines
   - Initial agent prototype
2. **Core Agent Loop** (Week 3–5)
   - Scout + Critic workflow
   - Logging, replay buffer
3. **Scalable Pipelines** (Week 6–8)
   - Data ingestion + experiment configs
   - Training pipelines
4. **Evaluation Framework** (Week 9–10)
   - Benchmarks, automated eval loops
5. **Expansion** (post-10 weeks)
   - Quantum embeddings
   - Generalized Research Agent (new repo)
