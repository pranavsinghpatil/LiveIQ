# Contributing to MeshCards

Thanks for your interest in improving MeshCards!  
We welcome contributions in all areas:

- UI improvements
- LLM prompt refinement
- Retrieval logic
- Anki deck features
- Documentation and examples

---

## How to contribute

1. Fork the repo & create a new branch:
   ```bash
   git checkout -b feature/my-cool-improvement
2. Make your changes

3. Add tests or examples when possible

4. Submit a Pull Request with:

Clear title & description

Screenshots if UI change

Reasoning behind the change

We’ll review and iterate with you 🤝

Coding guidelines

Python: Black formatting

TypeScript: ESLint + Prettier

Maintain modular and documented code

Avoid vendor lock-in for LLM providers

Community

If you build a new deck type or model integration:
→ Add it to the docs so others can learn and build on top of it ✨
