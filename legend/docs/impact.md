# Impact
mainly:

MeshCards unlocks the true value of personal notes by turning them into **actionable memory** using spaced repetition.

Today:
- Notes become **archives**
- Flashcard creation is **painful**
- Visual learners are **ignored**

MeshCards gives every learner:
- A **personal memory assistant**
- Visual memory if they want it
- Zero-friction active recall

This directly improves:
- test performance (GRE, GATE…)
- research retention
- ML/CS concept mastery
- lifelong learning discipline

## 1. Why this project matters

Most learners (including me) have:
- Tons of notes (Notion, Obsidian, Google Docs, PDFs, screenshots).
- Very few **actual flashcards**.
- A constant feeling of “I should make Anki cards… later.”

Result:
- We re-read instead of **actively recalling**.
- We forget concepts just when they finally start to “click”.
- Our notes stay a **dead archive**, not a living memory system.

MeshCards is meant to change that by:
- Turning existing notes into **ready-to-use Anki decks (.apkg)**.
- Making spaced repetition a **default outcome** of taking notes, not an extra chore.

---

## 2. Who benefits

- **Students prepping for exams**  
  (GRE, GATE, university exams, ML/DS interviews, etc.)
- **Self-learners in AI / CS / math**  
  with heavy text-based resources.
- **Knowledge workers** who read research papers, docs, RFCs, etc.

Instead of:
> “I’ll create cards later” → never happens

The flow becomes:
> Take notes → pick topic → click “Generate Anki Deck” → start reviewing tonight.

---

## 3. Concrete outcomes

### For learners

- More **active recall** with almost zero friction.
- Flashcards tailored to:
  - Their own notes (not generic internet summaries).
  - Their difficulty level and background.
- Less time formatting cards, more time actually **learning**.

### For your portfolio / career

- Demonstrates:
  - Practical use of **RAG + LLMs**.
  - Integration with a real tool (Anki, a widely used open-source SRS).
  - Thoughtful UX around **study workflows**, not just “chat with your notes”.
- Can be showcased as:
  > “A system that converts personal knowledge graphs + notes into
  > personalized Anki decks using retrieval + LLMs.”
